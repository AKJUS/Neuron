Coming Soon!

Satori employs Community Mining. This means the token holders own the token and how it opperates. Here you'll be able to vote on how the Satori system opperates. For example, voters will decide if changes need to be made to the Minting Schedule, as well as modifying distribution rates according to various mining activities.

Voters will also decide which datastreams are sanctioned for mining and which are not. This is important because the token holders have the power to direct the attention of the entire Satori prediction network.
